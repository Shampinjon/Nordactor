# Requierments
Python3

# Installing
sudo apt install nordactor-1.1.deb
pip install PySide6 --break-system-packages

## Features
- Tabs
- Open and save files
- Warning before closing unsaved files
- Settings page
- Saving tabs
- Searching text fragments
- Auto checking updates
- Light, Dark, OLED and System themes
- Minimal animations

- Вкладки

- Поиск фрагментов текста
- Автоматическая проверка обновлений
- Системная, темная, светлая и OLED темы.
