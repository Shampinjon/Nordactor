# Nordactor
Nordactor is a simple text editor.

# Requirements
pip install pyside6

# Run
python3 main.py

## Features
- Tabs
- Open and save files
- Warning before closing unsaved files
- Settings page with theme switching
- Light, Dark, OLED and System themes
- Minimal animations

# Nordactor(Русская версия)
ВНИМАНИЕ: Программа изначально будет на английском языке. Поэтому если хотите, смените язык в настройках.

- Вкладки
- Открытие и сохранение файлов

- Настройки с возможностью сменить тему
- Системная, темная, светлая и OLED темы.
